# Find a Doc MCP Server

MCP (Model Context Protocol) server for [Find a Doc, Japan](https://www.findadoc.jp) - connecting people in Japan to healthcare services.

## About

This is the MCP server for Find a Doc, providing AI-powered tools for interacting with our healthcare provider data.

## Getting Started

### Prerequisites

- Node.js >= 22.12.0
- Yarn 4.x

### Installation

```bash
yarn install
```

### Development

```bash
yarn dev
```

### Build

```bash
yarn build
```

### Linting

```bash
yarn lint
yarn lint:fix
```

### Testing

```bash
yarn test
yarn test:watch
```
